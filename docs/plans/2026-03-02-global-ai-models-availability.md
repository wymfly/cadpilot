# 全球顶尖 AI 3D 模型与底层 AI 算法可用性分析

在之前的调研中，我们发现了 NVIDIA、Stability AI 以及学界的一系列“降维打击”级黑科技。针对“它们是否真正开源可用”的疑问，以下是最新（截至 2025/2026 年）的落地状态调查报告：

---

## 1. 欧美 SOTA 生成模型的可用状态

欧美巨头的模型虽然强大，但在开源和 API 开放策略上有着严格的控制。

### NVIDIA Meshtron (最强工业级无损拓扑网格)
*   **开源状态**: **尚未开源 (Research Phase)**。
*   **可用性**: 目前仅发布了论文 (arXiv:2412.09548) 和演示视频。官方 GitHub 仓库还未放出代码或权重。
*   **预测**: 按照 NVIDIA 的惯例，这类高质量模型很可能最终不会提供可以直接免费商用的权重，而是被集成到 **NVIDIA NIM (微服务 API)** 或 Omniverse 中收费调用。

### NVIDIA Llama-Mesh (LLM 融合 3D)
*   **开源状态**: **完全开源，但限制商用**。
*   **可用性**: 代码和模型权重（基于 Llama-3.1-8B 微调）已在 GitHub (`nv-tlabs/LLaMA-Mesh`) 和 Hugging Face 上公开可用。
*   **商用限制**: 使用的是 **NSCLv1 (Non-Commercial Use Only) 协议**。仅限学术和个人研究，绝对不能直接集成到我们的商业 SaaS 流程中。

### Stability AI - SPAR3D (最速生成器)
*   **开源状态**: **有限开源 (Gated Model)**。
*   **可用性**: 代码已在 GitHub (`Stability-AI/stable-point-aware-3d`) 开源，模型权重在 Hugging Face 上可用，但需要同意许可协议。它甚至已经有了本地运行的 ComfyUI 节点。
*   **商业/API 状态**: 
    *   **本地商用**: 年收入 **低于 100万美元** 的团队可以免费商用。
    *   **官方 API**: Stability AI 的开发者平台 (API) 已提供调用，按次计费，非常适合直接接入我们的管线。
*   **结论**: **这是目前全球范围内最切实可行的商业化选项**。

---

## 2. 替代传统算法的深度学习开源项目

那些用来替代传统 C++ 算法（如水密化修复）的学界神经网络模型，大部分是完全开源的（MIT 协议），可以直接嵌入我们的 Python 环境。

### Neural-Pull (替代体素化修复，直接生成绝对水密 SDF)
*   **开源状态**: **完全开源 (MIT 协议)**。
*   **可用性**: 官方 PyTorch 版本 (`mabaorui/NeuralPull-Pytorch`) 在 GitHub 上可以直接克隆运行。
*   **适用场景**: 它接受任何极其破碎的点云或网格顶点，通过学习梯度的“拉力”，自动输出一个完美的闭合表面。
*   **结论**: 极高价值的开源资产，可以作为 `mesh_healer` 子图的终极杀手锏。

### DeepSDF (Facebook Research 开源补洞神器)
*   **开源状态**: **完全开源 (MIT 协议)**。
*   **可用性**: GitHub (`facebookresearch/DeepSDF`) 上有完整代码。
*   **适用场景**: 非常适合用来做“Shape Completion”（把切掉一半的残缺模型补全）。缺点是需要用 C++ 编译数据预处理模块，部署稍显繁琐。

### 神经布尔与神经切片 (Neural CSG / GNN Slicer)
*   **开源状态**: 大部分处于高校实验室的论文附带代码阶段（如 UCSG-NET）。
*   **可用性**: 属于“有源码但很难用”的状态。通常依赖过时的 TensorFlow 版本或缺乏文档，尚未形成类似 `manifold3d` 这种可以直接 `pip install` 的成熟工业级库。
*   **结论**: 现阶段（Phase 1/2）还是应该老老实实调用 C++ 的 `manifold3d` 做布尔运算，以及调用 CuraEngine CLI 做切片。

## 终极选型总结与回答

回答您的疑问：**这些最先进的模型，到底能不能用？**

1. **顶级生成网络**：NVIDIA 的王炸（Meshtron）还藏在实验室里，Llama-Mesh 禁止商用。当前**真正既开源又能商用、且素质顶级的全球化模型，首推 Stability AI 的 SPAR3D**（也可以直接调它的 API，成本极低）。
2. **底层几何替代神经网络**：用于“网格水密化修复”的 **Neural-Pull** 是完全开源、可本地部署、可自由修改的。我们完全可以把这个模型塞进 `mesh_healer` 节点里，取代那些动不动就导致内存溢出 (OOM) 的传统 Python 算法库。

这就形成了一个极其务实且前沿的落地方案：
*   **大阶段调度**: LangGraph
*   **艺术生成生坯**: 调用 SPAR3D API (速度快、规避版权、不吃本地显存)
*   **机械生成生坯**: LLM + CadQuery
*   **生坯破洞修复**: 在本地加载开源的 Neural-Pull 模型进行 SDF 重建。
*   **混合布尔装配**: `manifold3d` (最稳妥的 C++ 方案)。
